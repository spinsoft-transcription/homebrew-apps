# Homebrew Tap for Spinsoft Applications

## Installation

```bash
brew tap spinsoft-transcription/apps
brew install sign-language-recorder
```

Or install directly:

```bash
brew install spinsoft-transcription/apps/sign-language-recorder
```

## Update

```bash
brew upgrade sign-language-recorder
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `sign-language-recorder` | Sign Language Video Recording and Labeling Application |
